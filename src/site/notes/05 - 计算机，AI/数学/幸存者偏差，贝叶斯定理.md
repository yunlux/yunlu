# 抛硬币问题

今天麦上来了一个小伙伴问了个问题：

假设A一直在沉睡，旁边守着的B每天抛硬币，如果是正面，就只在周一唤醒B，如果是反面，就在周一至周日都唤醒B。补充条件：A每次醒来都不会记得之前发生的事，现在问你作为A醒来后猜测B所抛硬币的正反面

Claude3.5：

这是一个有趣的概率问题。让我们一步步分析:

1. 首先，我们需要明白A醒来的情况:
    - 如果硬币是正面，A只在周一醒来。
    - 如果硬币是反面，A在周一到周日都会醒来。
2. A每次醒来都不记得之前发生的事，所以A无法通过自己的记忆来判断。
3. 关键在于，A醒来时，更有可能是在硬币为反面的情况下。因为反面的情况下，A有7天的机会醒来，而正面只有1天。
4. 让我们计算概率:
    - 硬币正面的概率是1/2，此时A只有1/7的几率在这一天醒来。
    - 硬币反面的概率是1/2，此时A有7/7的几率在任一天醒来。
5. 使用贝叶斯定理，我们可以计算A醒来时硬币为反面的概率: P(反面|醒来) = P(醒来|反面) * P(反面) / \[P(醒来|反面) * P(反面) + P(醒来|正面) * P(正面)] = (7/7 * 1/2) / \[(7/7 * 1/2) + (1/7 * 1/2)] = 0.5 / (0.5 + 0.0714) ≈ 0.875 或 87.5%

因此，A醒来后应该猜测B所抛的硬币更可能是反面。具体来说，硬币为反面的概率约为87.5%（7/8），而为正面的概率约为12.5%（1/8）。

# 幸存者偏差

## 飞机弹孔

这个问题应该是个经典的概率论问题了

二战时期的一个真实故事，根据飞机身上的弹孔来加强设计。

看幸存的飞机，机翼上的弹孔多，发动机上的弹孔少，所以有人直觉会认为应该加强机翼的设计。但实际上，机翼弹孔少而发动机弹孔多的飞机已无法返航，我们能看到的就是幸存者。

# 结论

通过已有的样本，也就是结果，来研究某个规律，就应该担心落入幸存者偏差，没有**考虑周全**。

当我们选择已有的样本时，就只看到了幸存者，而忽略了没被统计到的样本。

比如，

查看公司财务报表时，就已经过滤掉了那些经营不善而破产的公司;
查看老年人在医院的诊疗记录时,就默认地排除了没有活到老年的人群;
统计某款手机软件的受欢迎程度，不自觉地排除了那些买不起手机或者从未安装过该手机软件的人。

这样的例子比比皆是。

之所以会产生幸存者偏差，是因为很多人从一开始就搞错了统计样本，只看到经过筛选的数据，但没有意识到筛选的过程。如果只是人为地选择部分观察数据，那就无法保证结论的客观性。

要获得”全样本”数据绝非易事。由于**认知局限**，很多人只看见了那些能看见的现象一比如受损的 飞机、就医的病人、成功的企业家、公司的报表，但忽略了没有看见的真相一未返航的飞机、健康的人、失败的创业者、破产的公司,而这些被忽略的数据同样重要,甚至更加重要。
