# 蛋炒饭

中国大陆网民反对中国共产党和中国政府的网络抗议方式，用以讽刺毛泽东长子毛岸英之死。抗议于每年10月24日（毛岸英的生日），又或者是11月25日（毛岸英的死亡日期）前后发生。网民以张贴蛋炒饭食谱的形式，来影射毛岸英于朝鲜战争期间据称因制作蛋炒饭而遭空袭身亡的事迹。有关贴文通常会被中共中央网信办封锁或删除，并会对涉事人士作出处罚。
